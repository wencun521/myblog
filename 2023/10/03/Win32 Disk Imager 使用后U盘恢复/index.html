Win32 Disk Imager 使用后U盘恢复

此文不仅仅处理标题的问题，对于所有U盘的恢复初始化都可以。

U盘在使用 win32 disk imager 写入过引导后，如果需要恢复U盘原始状态（初始化），可以按下面步骤操作：
1、windows下进入cmd命令窗口
2、输入 diskpart，显示如下：

C:\Users\Administrator>diskpart

Microsoft DiskPart 版本 10.0.14393.0

Copyright (C) 1999-2013 Microsoft Corporation.

在计算机上: SHANHY-PCDISKPART>

3、输入命令 list disk，显示如下：

DISKPART> list disk

磁盘 ### 状态 大小 可用 Dyn Gpt ——– ————- ——- ——- — — 磁盘 0 联机 465 GB 0 B * 磁盘 1 联机 7540 MB 7489 MB *

DISKPART>

4、很容易可以识别出U盘（此时我的U盘是磁盘1）输入 select disk 1 选择磁盘1，显示如下：

DISKPART> select disk 1

磁盘 1 现在是所选磁盘。

DISKPART>

5、输入 clean 清除磁盘（U盘中如果有重要内容记得提前备份），显示如下：

DISKPART> cleanDiskPart 成功地清除了磁盘。

DISKPART>

6、输入 create partition primary 创建主磁盘分区
7、输入 active 激活磁盘分区
8、输入 format fs=fat32 quick 以fat32格式快速格式化磁盘分区
这几步操作显示如下：

DISKPART> create partition primaryDiskPart 成功地创建了指定分区。

DISKPART> activeDiskPart 将当前分区标为活动。

DISKPART> format fs=fat32 quick 100 百分比已完成

DiskPart 成功格式化该卷。

DISKPART>
